# Vima Studio

Architecture & interior design studio website — a multi-page, bright-themed site with 3D/WebGL motion.

## Pages
- `index.html` — Home (WebGL hero, featured work, process)
- `about.html` — Studio & leadership
- `portfolio.html` — Filterable portfolio + lightbox
- `services.html` — Services, **Live Design Studio** (raw→final scrub) and **Constructive Design** build reel
- `contact.html` — Enquiry form + WhatsApp

## Stack
Static HTML/CSS/JS. Three.js (CDN) powers the hero scene. No build step.
